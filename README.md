# AskQueries

AskQueries is a CLI tool.

In AskQueries, the user enters context. Then, you can ask a few questions about the document.

AskQueries is already pre-trained on large datasets of technical documents, scientific papers, etc. This makes AskQueries perfect for quickly extrapolating answers from large documents.

## Dependencies

You need to have Python and pip installed.

```bash
# Install Pytorch
pin install torch 
# Install transformers (old) - prev versions do not support the bart functionalities
pip install transformers<5
```
## Running

```bash
python3 main.py
```

## Training

AskQueries was trained extensively with various collections of academic papers and programming documentation.
